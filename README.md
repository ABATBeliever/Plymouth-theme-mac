## 🖥️ 概要 / Overview

**Plymouth-theme-mac** は、UbuntuなどのLinuxディストリビューションで使用できる 
**macOS風のブートスプラッシュ（起動画面）テーマ**です。  

初めてのブートスプラッシュテーマです。

---

## ✨ 特徴 / Features

- macOSスタイルのApple風ロゴ
- Ubuntu (22.04LTS / 24.04LTS) / Lubuntu で動作確認 
- White / Black の2テーマ

---
